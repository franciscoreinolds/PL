Criamos uma makefile com vista a facilitar a compilação do programa, 
utilizando o comando: "make".

Após correr o "make", para executar o programa, escrever no terminal:

"./prog folha8.txt 100000",

onde folha8.txt deve ser substituído por qualquer outro ficheiro do
género que se pretenda que seja processado e o 100000 o número de artigos
do ficheiro alvo de processamento que se pretende processar (foi escolhido
100000 de modo a que se processasse o ficheiro por inteiro).

Após a execução do programa, para limpar os ficheiros originados como
resultado do processamento, correr o comando: "make clean".
